\fnsymbol

\fnsymbol{counter}

The \fnsymbol command causes the current value of counter to be printed in a specific sequence of nine symbols that can be used for numbering footnotes. (The sequence is asterisk, dagger, double dagger, section mark, paragraph mark, double vertical lines, double asterisks, double daggers, double double daggers.)

The value of counter must be less between 1 and 9.

Can be used only in math mode.
